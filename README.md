# ЗАРЕЧЬЕ — демо-сайт резиденц-клуба (real estate)

Премиальный **многостраничный** демо-сайт вымышленной компании — для показа
потенциальным клиентам как пример «уже сделанной работы». Русскоязычный,
с кинематографичными анимациями и motion design в стиле thebendclub.com.

## Как открыть

**Вариант 1 — двойной клик** по `Открыть сайт.bat` (запустит сервер и откроет браузер),
либо просто откройте `index.html`.

**Вариант 2 — вручную:**
```
cd "realestate demo"
python -m http.server 5173
```
→ http://localhost:5173/index.html

## Эффекты и анимации

- Кинематографичные **переходы между страницами** (зелёная шторка с брендом)
- Прелоадер со счётчиком · hero-карусель с Ken Burns · параллакс
- Проявление текста слово-за-словом · анимированные счётчики · бегущая строка
- Горизонтальный pin-скролл галереи · clip-reveal фото · масштаб-параллакс
- **Лайтбокс** (стрелки, клавиатура, подписи) · **фильтры** категорий
- Кастомный курсор · магнитные кнопки · плавный скролл (Lenis) · автоскрытие шапки

## Как адаптировать под клиента

- **Тексты / названия / цены** — в `.html` файлах (всё на русском).
- **Данные резиденций** (R1…R6: название, площадь, цена, описание) — объект `DB`
  в `main.js`, функция `residenceData`.
- **Фотографии** — заменить файлы в `assets/img/` (имена сохранить).
- **Цвета / шрифты** — CSS-переменные вверху `styles.css` (`--cream`, `--forest`, `--gold`).

## Деплой на GitHub Pages

Все фото и библиотеки **уже лежат в папке** `assets/` — ничего не грузится с внешних
ссылок (внешние только шрифты Google). Чтобы картинки точно отобразились:

1. Загрузите в репозиторий **содержимое** этой папки так, чтобы `index.html` и папка
   `assets/` оказались **в корне репозитория** (не внутри вложенной папки со страницей).
2. Обязательно закоммитьте папку `assets/` целиком (`assets/img/` и `assets/vendor/`).
3. Файл `.nojekyll` уже добавлен — он отключает Jekyll, чтобы GitHub Pages отдавал
   папки с ассетами как есть.
4. Settings → Pages → Source: ветка `main`, папка `/ (root)`.

> Частые причины «фото не грузятся на Pages»: папку `assets/` не закоммитили,
> либо несовпадение регистра в имени файла (Linux-сервер чувствителен к регистру).
> В этом проекте регистр и пути уже выверены — проверено.

## Технологии

Чистый HTML / CSS / JS. GSAP + ScrollTrigger (анимации), Lenis (скролл),
SplitType — всё локально в `assets/vendor/` (интернет нужен только для Google Fonts).

> Фотографии — стоковые (Unsplash, свободная лицензия). Все данные, имена, цены,
> телефоны и адреса **вымышленные** — это демонстрационный проект.
